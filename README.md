<h1>Activity Tracker</h1>
<h3>Log tasks as you complete them. Tracks name, category, and time spent. Saves locally to CSV.</h3>

<p>Uses file handling and CSV data storage to keep things simple and portable.</p>

<p><b>Features:</b></p>
<ul>
  <li>Log one task at a time, run and exit</li>
  <li>Categorize entries your way</li>
  <li>Track time spent per task (HH:MM)</li>
  <li>Local storage via CSV</li>
  <li>Read logged tasks</li>
  <li>Edit logged tasks</li>
</ul>
<p><b>Tech Stack:</b> Python</p>
